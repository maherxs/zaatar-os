# Changelog

All notable changes to Zaatar OS will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Complete Arabic language support with Syria (Damascus) locale as default
- Modern GTK theme, icons, and cursor themes
- Pre-installed GNOME extensions (Blur My Shell, Dash to Dock, Just Perfection, etc.)
- High-quality Arabic fonts (Cairo, Amiri, Tajawal, Noto Sans Arabic)
- Arabic language packs for Firefox and LibreOffice
- Arabic spell checking tools (hunspell-ar, aspell-ar)
- Pre-configured system settings optimized for Arabic users
- Automated build workflow with weekly scheduled builds
- Comprehensive documentation (README, CONTRIBUTING, LICENSE)

### Changed
- Default locale set to `ar_SY.UTF-8` (Syria - Damascus)
- Default timezone set to `Asia/Damascus`
- Arabic keyboard layout set as primary input source
- Dark theme enabled by default

### Fixed
- All scripts properly formatted with English comments
- Consistent code style across all files
- Proper error handling in installation scripts

## [1.0.0] - 2025-02-17

### Added
- Initial release of Zaatar OS
- Base system built on Aurora
- Complete Arabic localization
- Modern, elegant theming
- Pre-installed essential applications
